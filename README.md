## 会议室管理系统
#### 开发平台 
 - [x] 开发工具 IntelliJ Idea2016  MySql5.2 
 - [x] 框架技术 spring springmvc hibernate jpa  maven 
#### 实现功能  
 - [x] 会议室 会议 申请 用户 角色等等的管理
 - [x] 角色菜单管理 url过滤
 - [x] 申请使用会议室 审核申请 
 - [x] 会议记录 会议通知 查看历史会议
